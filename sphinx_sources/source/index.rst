========================================
Resources for Pyhton Certificate Program
========================================

Assorted Resources for the UW Python Certificate program:

https://www.pce.uw.edu/certificates/python-programming

This page contains various resources for students in the University of Washington Professional and Continuing Education Python Certificate Program.

For the most part, these are resources of use to any python programmer: book recommendations, links to tutorials, blog posts, etc. If you see any outdated links, errors, or have suggestions for additions, please don't hesitate to fork the repo at: https://github.com/UWPCE-PythonCert/PythonResources and produce a pull request for us to review.
